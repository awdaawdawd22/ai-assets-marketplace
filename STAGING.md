# Стадия и тестовый доступ

## Стадия
- `development`
- Приложение работает как прототип.

## Тестовый доступ
- Локальный URL: `http://localhost:3000`
- Админ-панель: `/admin`
- Тестовый пароль: `admin123`
- Личный кабинет: `/account` (переключатель роли Покупатель/Продавец)

## Что можно протестировать
- Магазин активов на главной странице `/`
- Выбор ассета и переход к подробному описанию
- Оформление заказа через `/checkout`
- Админ-панель:
  - редактирование цены
  - включение/выключение `On sale`
  - переключение флага `Featured`
  - просмотр заказов на `/admin/orders`

## Ограничения текущей реализации
- Админка работает только на клиенте и хранит состояние в `localStorage`.
- Пароль зашит в коде в виде демо-данных и не шифруется.
- Отдельного кабинета продавец/покупатель с переключателем пока нет.
- Это рабочий прототип для локального теста, не для боевого деплоя.

## Быстрый запуск
```bash
pnpm install
pnpm dev
```

Затем откройте `http://localhost:3000` и перейдите на `/admin`.
