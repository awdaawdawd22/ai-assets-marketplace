# AI Assets Marketplace

Premium digital assets marketplace built with Next.js 16, Tailwind CSS, and shadcn/ui.

## Features
- Modern dark UI
- Asset browsing and detailed views
- Search and filtering
- Shopping cart
- Reviews system

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Zustand
- SWR
- Lucide Icons
